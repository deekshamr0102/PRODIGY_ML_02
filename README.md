# Customer Segmentation using K-Means Clustering

## 📌 Project Overview
Customer segmentation is an important technique used by businesses to understand their customers better.  
This project uses the **K-Means Clustering algorithm** to group customers based on their purchasing behavior.

By identifying different customer groups, companies can design better marketing strategies and improve customer satisfaction.

---

## 🎯 Objective
The objective of this project is to:
- Analyze customer purchase data
- Group customers into different segments
- Visualize customer clusters using machine learning techniques

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab / VS Code

---

## 📊 Machine Learning Algorithm
This project uses the **K-Means Clustering algorithm**, an unsupervised machine learning technique that groups similar data points together based on their features.

Steps followed:
1. Data Loading
2. Data Preprocessing
3. Feature Selection
4. Applying K-Means Clustering
5. Visualizing the Clusters

---

## 📁 Dataset
The dataset contains customer purchase information such as:
- Customer ID
- Annual Income
- Spending Score

This data is used to group customers with similar behavior.

---

## 📈 Output
The model creates clusters of customers and visualizes them using scatter plots to identify patterns in customer spending behavior.
